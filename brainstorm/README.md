# brainstorm

Design history, proofs, notes, and decision records for MIWN. Working material —
not the manuscript. Curated keepers (proofs worth preserving, key derivations)
live here; ephemeral scratch does not.

- `decisions/` — Architecture Decision Records (ADRs), one per file, numbered.
  Start at `decisions/0001-repo-structure.md`.
- `rezn/` — curated proofs, derivations, and design notes migrated from the
  predecessor REZN repo (@ `7f03509`): theory, annotated proofs, the system
  equations, the posterior method, and the extended proofs appendix.
